# Stream Dream
  Stream Dream is my first backend application, developed to explore the fundamentals of building a media management system.
This REST API allows users to organize their movies and watchlists in a structured way.

# Tech Stack
Language: Java 21
Framework: Spring Boot
Build tool: Maven
Database: MySql

# Features
This project is currently a work in progress. As my first major coding project, I am focusing on:
External API Integration: Fetching movie data from a third-party API to populate the database.
Core Entities: Implementing structures for Users and Media content.
REST Endpoints: Developing routes for viewing, searching, and adding media entries.
Database Management: Establishing a reliable connection with a MySQL database.

# Setup Requirements
To explore or run the code locally:
-Java 21 JDK and Maven are required.
-A local MySQL instance must be configured.
-A personal application.properties file must be created in src/main/resources/ with your specific database credentials.
